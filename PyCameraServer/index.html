PyCameraServer

PyCameraServer is a PyCameraServer is a Flask video / image / Youtube / IP Camera frames online editor with live streaming preview for objects recognition, extraction, segmentation, resolution upscaling, styling, colorization, interpolation, using OpenCV with neural network models: YOLO, Mask R-CNN, Caffe, DAIN, EDSR, LapSRN, FSRCNN, ESRGAN.
